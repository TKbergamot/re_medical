# Re:medical

## サイト概要

    - 全国の病院でのセカンドオピニオンを受けれるサイト

### サイトテーマ

    - 全ての患者様に選ぶ自由を。

### テーマを選んだ理由

    - 来診される患者様が近場の病院以外でもいろいろなセカンドオピニオンを受け、納得して医療を受けれるようにしたい。

### ターゲットユーザ

    - 来院される患者様、またはご家族の方。

### 主な利用シーン

    - 患者様がセカンドオピニオンを希望した時や、医師が判断した時など。
